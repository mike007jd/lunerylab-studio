# Luna Studio DNA Brief

This is the visual and interaction DNA for Lunery Lab UI work.

## Product Feeling

Lunery Lab should feel like a focused local-first creative studio, not a SaaS
dashboard, not a generic chat app, and not a model manager.

The interface is dense enough for repeated creative work, but quiet enough that
the generated artwork stays visually dominant.

## Visual Identity

- Matte dark surfaces.
- Moon-white text.
- Silver primary actions.
- Restrained gold accent for provenance, capability, and high-signal status.
- Crisp borders instead of heavy shadows.
- Compact cards and controls.
- No decorative gradient blobs, no generic glassmorphism, no marketing-style
  hero treatment inside the desktop Studio.

## Layout Grammar

- Studio is the primary workspace.
- Settings is a capability and runtime layer, never the main product.
- Canvas supports artifact editing and context, but must not swallow the product
  into a blank generic board.
- Assistant is a helper surface. Direct action stays primary where possible.

## Component Grammar

- Buttons use local shadcn `Button`.
- Status chips use local `Badge`.
- Page rhythm uses design-system shell helpers.
- Surface text uses compact hierarchy: caps labels, short headings, concise
  helper copy.
- Icons come from the repo icon wrappers.

## Motion Grammar

- Motion should explain state or preserve spatial continuity.
- Use the named timing tiers: `micro` 120ms, `control` 160ms, `overlay` 200ms,
  `modal` 240ms, `surface` 260ms, and `exit` 160ms.
- Arrivals decelerate; departures accelerate, and exit never outlasts entry.
- Components consume semantic presets or CSS motion tokens rather than numeric
  duration/easing literals.
- Springs are reserved for direct manipulation.
- No decorative ambient animation in utility surfaces.
- Respect reduced motion.

## Copy Grammar

- Avoid SaaS billing, teams, credits, pro-tier, enterprise, and account language.
- Avoid narrow e-commerce/product-listing framing unless a feature explicitly
  earns that vertical.
- Speak as a creator tool: local, capable, direct, and calm.
